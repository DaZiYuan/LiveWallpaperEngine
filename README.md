# LiveWallpaperEngine 巨应壁纸引擎

## 功能：
Windows10 动态壁纸极简API

## 语言框架：
DotNet

## 功能目标：
- [x] 嵌入桌面
- [x] exe壁纸鼠标事件转发 (现已支持，感谢[ADD-SP](https://github.com/ADD-SP)的PR)  
- [x] 支持多屏幕
- [x] 内置视频渲染
- [ ] 内置web渲染 （已初步完成，多屏还有bug）
- [x] 无UI壁纸引擎
